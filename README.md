#### 一、功能介绍
1. **支持直接解析标准URL进行跳转，并自动注入参数到目标页面中,目前版本只支持activity**
2. **支持多模块工程使用**
3. **支持依赖注入**


#### 二、典型应用
1. 跨模块页面跳转，模块间解耦
2. 跨模块API调用，通过控制反转来做组件解耦
